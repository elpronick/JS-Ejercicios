# Practica de JavaScript Vanilla

Proyecto de entrega con 9 ejercicios independientes para practicar JavaScript,
manipulacion del DOM, eventos y `localStorage`.

## Estructura

```text
.
|-- index.html
|-- js_ejercicio_1.html
|-- js_ejercicio_2.html
|-- js_ejercicio_3.html
|-- js_ejercicio_4.html
|-- js_ejercicio_5.html
|-- js_ejercicio_6.html
|-- js_ejercicio_7.html
|-- js_ejercicio_8.html
|-- js_ejercicio_9.html
|-- css/
|   `-- style.css
|-- scss/
|   `-- style.scss
`-- js/
    |-- js_ejercicio_1.js
    |-- js_ejercicio_2.js
    |-- js_ejercicio_3.js
    |-- js_ejercicio_4.js
    |-- js_ejercicio_5.js
    |-- js_ejercicio_6.js
    |-- js_ejercicio_7.js
    |-- js_ejercicio_8.js
    `-- js_ejercicio_9.js
```

## Ejercicios

1. Cambiador de color de fondo.
2. Contador de clics.
3. Lista dinamica.
4. Filtro en tiempo real.
5. Calculadora sencilla con validaciones.
6. Temporizador.
7. Generador de contrasenas.
8. Contador de palabras y caracteres.
9. Lista de tareas persistente con `localStorage`.

## Como usarlo

Abre `index.html` en el navegador y entra en cada ejercicio desde el indice.

Si quieres recompilar el SCSS manualmente y tienes Sass instalado:

```bash
sass scss/style.scss css/style.css
```
